Privacy Policy

Last updated: July 9, 2026

Beta notice: This is a practical starter privacy policy for the Chalked beta and should be reviewed before a broad public launch.

Information We Collect

We collect account details such as username, email, display name, password hash, profile images, league memberships, picks, standings, chat messages, feedback reports, and settings you create in Chalked.

Technical Data

We may store session data, IP address, browser user agent, error details, upload metadata, and server logs to secure the service, debug issues, prevent abuse, and operate the product.

How We Use Information

We use information to run leagues, authenticate users, send verification and password reset emails, settle matchups, show leaderboards, moderate accounts, improve the product, and respond to support requests.

Sharing

We do not sell personal information. We use service providers for hosting, email delivery, object storage, analytics or operations, and sports data. Public league activity, standings, display names, avatars, and chat may be visible to other league members.

Your Choices

You can update your profile in the app. You can leave leagues, request support, and ask for account help through the report issue flow or support email.

Security

We use hashed passwords, sessions, rate limits, backups, and admin tools. No online service is perfectly secure, so use a unique password and report suspicious behavior.

Contact

For privacy or support requests, email support@playchalked.com.
